- 64卦新增：核心摘要、关键词、一句话卦意、适用场景、行动建议、风险提醒。

## 内容原则

- 不复制现代网站解释文字。
- 应用层内容为原创整理。
- 以《周易》卦辞、爻辞、彖传、象传和卦象结构为基础。
- 现实应用采用“提醒、观察、行动建议”的方式，不做绝对化预测。
