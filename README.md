## 重要说明

本版“经典注解合参”是学习型原创摘要，不是王弼、程颐、朱熹原文逐字引文。它的作用是帮助你理解不同注解传统常关注的角度：

- 王弼：义理、时义、主旨、得中得位
- 程颐：修身、处事、天理、人事责任
- 朱熹：本义、占问、卦爻文本与所处爻位
- 象数：上下卦、互卦、错卦、综卦所形成的结构观察
