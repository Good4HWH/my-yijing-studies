# 我的易经｜学习与数字卦 v0.8

这是一个私人中文 iPhone Web App / PWA，用于《易经》学习、数字卦起卦、个人笔记与卜卦记录。

## v0.8 更新重点

- 资料库结构升级：每卦支持：
  - 卦辞原文 + 白话译文
  - 彖传原文 + 白话译文
  - 象传原文 + 白话译文
  - 六爻爻辞原文 + 逐爻白话译文
  - 原文来源链接
  - 资料校对状态
- 已完整扩充第1—56卦：
  - 乾、坤、屯、蒙、需、讼、师、比
  - 小畜、履、泰、否、同人、大有、谦、豫

  - 无妄、大畜、颐、大过、坎、离、咸、恒
  - 遯、大壮、晋、明夷、家人、睽、蹇、解
  - 损、益、夬、姤、萃、升、困、井
  - 革、鼎、震、艮、渐、归妹、丰、旅
- 保留完整64卦卦序、卦名、卦图、上下卦结构与数字卦计算。
- 其余第57—64卦暂保留为“待精校补齐”，避免使用未经核验的译文充数。

## 资料原则

资料库优先级：

1. 原典准确
2. 来源可追溯
3. 白话解释原创整理
4. 逐卦逐爻精校
5. 不直接复制当代出版社译文或不明版权内容

## 重要说明

此版本是“增强版第七包”，不是最终校定版。为了资料质量，未校对的卦不会伪装成已完成。后续版本应继续补齐第9—64卦的原典与译文。

## 安装

上传到 GitHub Pages 后，用 iPhone Safari 打开链接，选择“分享 → 添加到主屏幕”。

## 备份

卜卦记录与学习笔记保存在本机浏览器 localStorage。请定期用“导出备份”保存 JSON 文件。
