# 我的易经｜学习与数字卦 v1.0 精修版

私人中文 iPhone Web App / PWA，用于《易经》学习、数字卦起卦、个人笔记与卜卦记录。

## v1.0 精修重点

- 64卦资料库已完成首轮全量结构化。
- 每卦包含：
  - 卦辞原文与白话译文
  - 彖传原文与白话译文
  - 象传原文与白话译文
  - 六爻爻辞与逐爻白话译文
  - 白话学习提示
  - 来源链接
  - 校对状态
- 新增与优化：
  - 收藏卦
  - 全文检索：卦名、卦序、卦辞、爻辞、译文、关键词
  - 今日一卦
  - 数字卦结果页重排
  - 本卦 / 动爻 / 变卦结构化展示
  - 起卦结果复制与分享
  - 卜卦记录搜索
  - 备份导出内容包含记录、笔记、收藏
  - 先天八卦图 App 图标继续保留

## 安装

上传到 GitHub Pages 后，用 iPhone Safari 打开链接，选择：

分享 → 添加到主屏幕

如果旧版本缓存未刷新，请删除主屏幕旧图标后重新添加。

## 备份

卜卦记录、学习笔记与收藏存储在本机浏览器 localStorage。请定期使用“导出备份”。
