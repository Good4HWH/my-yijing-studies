# 我的易经｜学习与数字卦 v1.3A

本版落实首页、64卦列表和详情返回导航的界面改进，同时保留 v1.2 Final 全部64卦资料。

## v1.3A 改进

- 首页保持简洁：
  - 保留 App 图标与标题
  - 三个主入口改为更大、更清楚的按钮
  - 版本说明和更新记录移到“信息”页
- 64卦列表：
  - 卦象放大约 50%
  - 卦意摘要字体放大约 25%
  - 卦象使用固定宽度，滚动时保持对齐
  - 移除每卦旁边的“完整”标签
- 详情导航：
  - 从数字卦结果打开本卦或变卦后，可返回原来的数字卦结果
  - 从首页、64卦、记录或信息页进入详情时，返回按钮会回到对应来源
  - 在详情页继续浏览互卦、错卦、综卦时，仍保留原始返回来源
- 信息页：
  - 集中版本说明、资料原则、算法、备份和未来 AI 方向
- 质量检查：
  - 验证64卦数量与卦序
  - 验证64种上下卦组合无重复
  - 验证每卦六爻、六条译文、核心原典、来源和经典核引状态

## AI 分析路线

未来 AI 功能可结合：

- 用户的具体问题与背景
- 本卦、动爻、变卦
- 64卦原典和经典核引资料
- 现实应用、风险和行动建议

AI 应通过安全的后端服务调用，不能把 API key 放在 GitHub Pages 或浏览器代码里。
